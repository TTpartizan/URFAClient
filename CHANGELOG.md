## История изменений
**v2.1.0**
- Добавлена возможность выбора api.xml из директории xml, без указания полного пути
- Убрана зависимость от модуля bcmath

**v2.0.0**
- Убран параметр log
- namespaces PSR-4
- Функции возвращают ArrayObject
- Передача параметров функции в формате JSON

**v1.3.5**
- Обработка атрибута `long` перед `for` #40
- Исправлены ошибка соединения по протоколу `tls` для устаревших версий PHP

**v1.3.4**
- Обработка атрибута `set` в `input` #41

**v1.3.3**
- Исправления deprecated для PHP 7.4

**v1.3.2**
- Исправлен баг с пользовательскими функциями (user5)

**v1.3.1**
- Добавлен автоматический выбор протокола SSL соединения, работает с OpenSSL 1.1 и ядром UTM-5.3-005-update2

**v1.3.0**
- Доработан анализ узла `output`
- Добавлена обработка тэга `set`

**v1.1.0**
